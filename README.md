# Revii AI - Product Improvement Demo

**Author:** Paul Ng | **For:** Tom Ferry International

This demo showcases three key improvements to the Revii AI platform, demonstrating hands-on AI/LLM product building skills, cost optimization strategies, and technical leadership thinking.

## Demo 1: Smart Roleplay Scoring Engine
Enhanced AI roleplay system with multi-dimensional scoring, real-time feedback, and improvement tracking.

## Demo 2: RAG Pipeline for Real Estate Knowledge
Domain-specific knowledge retrieval system using vector embeddings for instant, accurate real estate answers.

## Demo 3: AI Agent Cost Optimizer
Intelligent model routing and caching system that reduces LLM API costs by 40-60% without quality loss.

## Quick Start

```bash
npm install
# Copy .env.example to .env and add your API keys
cp .env.example .env
node demo-roleplay.js
node demo-rag-pipeline.js
node demo-cost-optimizer.js
```
